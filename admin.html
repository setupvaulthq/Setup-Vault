Ürün siteye neden düşmüyor?

Bu panel bir veritabanına yazmıyor; sadece products.json üretir. Canlı site her zaman repodaki data/products.json dosyasını okur.

  1. Ürün Ekle → tarayıcı otomatik olarak güncel JSON dosyasını indirir.
  2. Aşağıdan GitHub’a kaydet (hesap ayarlıysa) → repoya yazar, Vercel yeniden deploy eder (~1–2 dk).
  3. Ya da indirdiğin dosyayı projede data/products.json üzerine yapıştırıp git push yap.
  4. Görsel dosyayı (.jpg) repo köküne ekleyip push etmeyi unutma.

Add New Product

ID = kısa dosya/anchor adı (harf-rakam-tire). Veritabanı “max 11” veya ASIN değil; boş bırakırsan isimden üretilir (~80 karakter üstü engellenir).

No image file

Önizleme = canlı sitede /dosya.jpg adresinden yüklenir. JPG’yi henüz GitHub’a push etmediysen veya deploy olmadıysa “bulunamadı” normaldir; ürün yine de JSON’a yazılır.

Paste an Amazon or amzn.to link.

Görsel seç → (isteğe bağlı) Image file name alanına repo kökündeki hedef adı yaz (ör. my-gear.jpg) → yükle. Boşsa dosya adından türetilir. Ürün eklerken bu alan otomatik dolar.
